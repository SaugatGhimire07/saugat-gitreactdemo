The purpose of this project is to demonstrate the use of Git and GitHub for version control and collaboration in a React Vite application. This project, named react-saugat, involves initializing a local Git repository, creating a new React app, and managing development through branching and merging strategies. 

Key features include the creation of a new component, and the use of GitHub for pushing and pulling changes to collaborate effectively. ZThis project serves as a hands-on exercise in applying DevOps practices in software development.
